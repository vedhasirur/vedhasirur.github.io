üå± Plant Water Monitoring System

I built this water level monitoring system to automate and improve water resource management efficiency. I implemented real-time monitoring and control by utilizing the NodeMCU with a relay module, integrating embedded systems and IoT technologies. I incorporated the ultrasonic sensor and mapped its readings to LED indicators and an I2C-enabled LCD display. I programmed the NodeMCU with code to measure the water level using the ultrasonic sensor. I set up both web and mobile dashboards to enable remote monitoring and control. In this video, I demo the LED indicators and LCD display displaying the current water level from low to full (it appears faint on video). A pump can be attached to automate watering via the relay module.
